Weather App

Status: Active Development

Skills Shown: Python Development, APIs, GUI & App Development, Good Coding Practices, Problem-Solving

Description: Weather App is a interactive and user-friendly project which retrieves data from OpenWeatherMap to learn about the current weather on your or any area in the world. It includes many details like temperature, pressure, wind speeds, visibility, and much more. On top of that, it contains custom climate alerts which can be useful (these alerts are custom-made and not an officially alert from the National Weather Services).

Weather Data Program

Status: Finished!

Skills Shown: Python Development, APIs, Graphs, Data Analysis, Databases

Description: This project is different from Weather App. It is a program which extracts NOAA data and creates a weather database with accurate present and next week details. It includes a histogram which provides visual data of humidity. While it might lack a couple features, it showcases advanced data analysis.
